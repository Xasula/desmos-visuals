Taylor Series Expansions

This visualization shows how a Taylor polynomial T(x) (in purple) approximates a function f(x) (in orange).

The Taylor expansion centered at x = c begins like this:

T(x) = f(c) + f'(c)(x - c) + f''(c)(x - c)2/2! + f'''(c)(x - c)3/3! + ...

The slider n controls how many terms of this expansion are inclued. When n = 1, T1(x) = f(c) + f'(c)(x - c) is just the linear approximation. As n increases, the approximation uses more derivative information, so the purple curve usually matches the orange curve more closely near x = c.

The point (c, f(c)) is the center of the approximation. The Taylor polynomial is built to match f(x) and its derivatives at this point, so the approximation is usually best near x = c and may become less accurate farther away.

THe error function R(x) = |T(x) - f(x)| gives the actual error of the Taylor approximation at a given location x. A smaller value of R(x) means the purple curve of T(x) has a smaller vertical distance to the orange curve of f(x) at x.

The Lagrange Error Bound gives a guaranteed upper bound for the error R(x). For a degree n Taylor polynomial centered at x = c, the bound is

R(x) ≤ M |x - c|n+1 / (n + 1)! ,

where M is the upper bound for the absolute value of the n+1th derivative of f between c and the chosen x-value.
