Why Limaçons Are Convex When |a/b| ≥ 2

Goal

We want to explain why the limaçon r = a + b cos(θ) becomes convex when |a/b| ≥ 2.

Intuitively, this condition means that the constant part a is large enough compared to the changing part b cos(θ). The changing part can still stretch the curve, but it is not strong enough to pull the curve inward and create a dimple.

Note: We can assume a > 0 and b > 0

The signs of a and b mainly affect the orientation of the graph. Changing signs can rotate or reflect the limaçon, but rotations and reflections do not change whether a curve is convex.

Thus, to understand the shape, it is enough to prove the result in the standard case where

a > 0,   b > 0,   and   a/b ≥ 2.

At the end, this becomes the usual absolute-value condition |a/b| ≥ 2.

Claim 1: r never becomes negative

Since cos(θ) is always between -1 and 1, the smallest possible value of r = a + b cos(θ) occurs when cos(θ) = -1.

rmin = a + b(-1) = a - b.

If a/b ≥ 2, then a ≥ 2b. Therefore,

rmin = a - b ≥ 2b - b = b > 0,

meaning r is always positive. The curve never crosses through the pole (r = 0) to form an inner loop.

Claim 2: The only possible dimple is on the left side (x ≤ 0)

Recall that we are working with a, b > 0.

b cos(θ) > 0 on the right side (x > 0), and b cos(θ) < 0 on the left side (x < 0). This is because cos(θ) = x/r, so b cos(θ) and x are either both positive or both negative. Thus, the curve is pushed outward when on the right of the y-axis and pulled inward when on the left of the y-axis.

Therefore, if a dimple appears, it must appear on the left side of the limaçon, where π/2 ≤ θ ≤ 3π/2.

Writing x and y in terms of θ

Convert the polar equation into parametric form:

x = r cos(θ) = (a + b cos(θ))cos(θ)
y = r sin(θ) = (a + b cos(θ))sin(θ).

Differentiate both with respect to θ and clean up:

dx/dθ = -a sin(θ) - b sin(2θ)
dy/dθ = a cos(θ) + b cos(2θ)

Why use the second derivative?

The possible dimple is on the left side of the curve. On that side, we can think of the curve as a sideways graph x = g(y). For a sideways graph, d2x/dy2 tells us whether the curve is "concave right" or "concave left."

On the left side of our limaçon, d2x/dy2 shall be greater than or equal to zero so that the curve is "concave right" (The curve must bend outward instead of forming an inward dimple).

Finding the second derivative

Since x and yo are both functions of θ, the second derivative d2x/dy2 should be

d2x/dy2 = d/dy(dx/dy) = d/dθ[(dx/dθ) / (dy/dθ)] · dθ/dy.

Since dθ/dy = 1/(dy/dθ), this becomes

d2x/dy2 = ((d2x/dθ2)(dy/dθ) - (dx/dθ)(d2y/dθ2)) / (dy/dθ)3.

Plugging in the expressions for x(θ) and y(θ), we get

d2x/dθ2 = -a cos(θ) - 2b cos(2θ)
d2y/dθ2 = -a sin(θ) - 2b sin(2θ).

Substituting and simplifying gives

d2x/dy2 = -(a2 + 3ab cos(θ) + 2b2) / (a cos(θ) + b cos(2θ))3.

Claim 3: The second derivative is never negative exactly when a/b ≥ 2

We are interested in the no-inner-loop case, so assume a > b. On the left side,

π/2 ≤ θ ≤ 3π/2.

In this interval, dy/dθ = a cos(θ) + b cos(2θ) is negative, so

(dy/dθ)3 < 0.

Therefore, d2x/dy2 ≥ 0 exactly when

a2 + 3ab cos(θ) + 2b2 ≥ 0

for every point on the left side.

The smallest value of this expression occurs when cos(θ) = -1. So we need

a2 - 3ab + 2b2 ≥ 0.

Factor:

a2 - 3ab + 2b2 = (a - b)(a - 2b).

Since we are already in the no-inner-loop case a > b, the factor a - b is positive. Therefore,

(a - b)(a - 2b) ≥ 0   exactly when   a - 2b ≥ 0.

Thus,

d2x/dy2 ≥ 0   exactly when   a/b ≥ 2.

Conclusion

When a/b ≥ 2, the second derivative d2x/dy2 is never negative on the left side. So the left side opens outward like x = y2, which means no dimple forms.

Since Claim 1 also showed that r never becomes negative, there is no inner loop. Therefore the limaçon is convex.

Since rotations and reflections do not change convexity, the general condition is

|a/b| ≥ 2.
