Limaçon Visualizer

Overview

This visualization shows how the polar curve r = a + b cos(θ) is drawn. As θ increases from 0 to 2π, the radius r changes depending on the value of a + b cos(θ).

The purple reference circle r = a shows the constant-radius part of the formula. The limaçon is formed by adding the changing term b cos(θ) to this base radius. When b cos(θ) > 0, r becomes larger than a. When b cos(θ) < 0, r becomes smaller than a.

Positive and Negative r

In polar coordinates, a positive r means the point is plotted in the usual direction of θ. In this visualization, those parts are shown in red.

A negative r means the point is plotted in the opposite direction, shifted by π radians. In this visualization, those parts are shown in blue. These negative-r parts are what create an inner loop when the changing term is large enough.

Interactive Graph

The graph displays reference values for the current angle θ = t. It shows the radius as r = a + b cos(θ), with the two parts displayed separately. It also labels the current value of r and the current angle θ, rounded to two decimal places.

Press play on t to see the limaçon being traced. The red parts show where r ≥ 0, and the blue parts show where r < 0.

How the Ratio |a/b| Changes the Shape

The shape of the limaçon depends on the ratio |a/b|. This ratio compares the constant part a with the changing part b cos(θ). In the usual case where ab ≠ 0, the size of this ratio determines which type of limaçon appears.

|a/b| < 1: Inner Loop Limaçon

|a| < |b|. This means b cos(θ) is strong enough to make r = a + b cos(θ) negative for some values of θ. Since points with negative r-values are plotted in the opposite direction, the curve forms an inner loop.

|a/b| = 1: Cardioid

If |a| = |b|, then the smallest radius is exactly 0. The curve touches the pole but does not pass through it to form an inner loop. This special limaçon is called a cardioid.

1 < |a/b| < 2: Dimpled Limaçon

If |b| < |a| < 2|b|, then r never becomes negative, so there is no inner loop. However, the changing term b cos(θ) is still large enough to pull one side of the curve inward, creating a dimple.

|a/b| ≥ 2: Convex Limaçon

If |a| ≥ 2|b|, then the constant part a dominates the changing term b cos(θ). The curve is not pulled inward enough to form a dimple, so the limaçon becomes convex.
