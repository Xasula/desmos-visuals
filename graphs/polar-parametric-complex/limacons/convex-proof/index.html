Why Limaçons Are Convex When |a/b| ≥ 2

Goal

We want to explain why the limaçon r = a + b cos(θ) is a convex if and only if |a/b| ≥ 2.

Intuitively, this condition means that the constant part a is large enough compared to the changing part b cos(θ). The changing part can still stretch the curve, but it is not strong enough to pull the curve inward and create a dimple.

Language note: “If and only if” means the statement works in both directions. Here, it means: if |a/b| ≥ 2, then the limaçon is convex, and if the limaçon is convex, then |a/b| ≥ 2.

Note: We can assume a > 0 and b > 0

The signs of a and b mainly affect the orientation of the graph. Changing signs can rotate or reflect the limaçon, but rotations and reflections do not change whether a curve is convex.

Thus, to understand the shape, it is enough to prove the result in the standard case where

a > 0,   b > 0,   and   a/b ≥ 2.

At the end, this becomes the usual absolute-value condition |a/b| ≥ 2.

Claim 1: r never becomes negative

Since cos(θ) is always between -1 and 1, the smallest possible value of r = a + b cos(θ) occurs when cos(θ) = -1.

rmin = a + b(-1) = a - b.

If a/b ≥ 2, then a ≥ 2b. Therefore,

rmin = a - b ≥ 2b - b = b > 0,

so r is always positive. The curve never touches or crosses through the pole, meaning it cannot form an inner loop.

Claim 2: The only possible dimple is on the left side

Remember that we are working with a, b > 0.

The term b cos(θ) is positive on the right side and negative on the left side. This is because cos(θ) = x/r. Since Claim 1 tells us r > 0, the sign of cos(θ) must match the sign of x.

When x > 0, the curve is pushed outward because b cosθ > 0. When x < 0, the curve is pulled inward because b cosθ < 0. Therefore, if a dimple appears, it must appear on the left side of the limaçon.

What you should notice: as b increases, the b cos(θ) term pulls the left side inward while pushing the right side outward.

Writing x and y in terms of θ

Convert the polar equation into parametric form:

x = r cos(θ) = (a + b cos(θ))cos(θ)
y = r sin(θ) = (a + b cos(θ))sin(θ)

Differentiate both with respect to θ and simplify:

dx/dθ = -a sin(θ) - b sin(2θ)
dy/dθ = +a cos(θ) + b cos(2θ)

Why use the second derivative?

If there ever is a dimple, it must be on the left side of the curve. Restricting our view to just the left (x < 0) part, we can think of the curve as a sideways graph x = g(y).

For a sideways graph, d2x/dy2 tells us whether the curve bends to the right or to the left. To avoid an inward dimple on the left side, the curve should bend outward, toward the exterior of the limaçon.

Try changing b. The blue parabola shows the local curvature at the leftmost point (θ = π). The sign of the second derivative d2x/dy2 changes when a = 2b.

Finding the second derivative

Since x and y are both functions of θ,

dx/dy = (dx/dθ) / (dy/dθ).

To find the second derivative, differentiate dx/dy with respect to y:

d2x/dy2 = d/dy(dx/dy).

Since dy/dθ is not constant, we differentiate the whole fraction (dx/dθ) / (dy/dθ). Using the quotient rule,

d/dθ[(dx/dθ) / (dy/dθ)] = ((d2x/dθ2)(dy/dθ) - (dx/dθ)(d2y/dθ2)) / (dy/dθ)2.

Also, dθ/dy = 1/(dy/dθ). Therefore,

d2x/dy2 = ((d2x/dθ2)(dy/dθ) - (dx/dθ)(d2y/dθ2)) / (dy/dθ)3.

Plugging in the expressions for dx/dθ and dy/dθ and simplifying gives

d2x/dy2 = -(a2 + 3ab cos(θ) + 2b2) / (a cos(θ) + b cos(2θ))3.

Claim 3: The second derivative is always nonnegative if and only if a/b ≥ 2

We do not want inner loops or cardioids, so assume a > b. The left side of the curve is drawn when

π/2 ≤ θ ≤ 3π/2.

On this interval, dy/dθ < 0. In other words, as θ increases, the left side of the limaçon is traced downward.

Note that when the current θ value increases from π/2 to 3π/2, the curve is drawn downward. Therefore, dy/dθ must be negative in the interval from π/2 to 3π/2.

Therefore,

(dy/dθ)3 < 0.

Because the denominator (dy/dθ)3 of d2x/dy2 is negative, the second derivative d2x/dy2 is nonnegative if and only if

a2 + 3ab cos(θ) + 2b2 ≥ 0

for every point on the left side.

The smallest value of this expression occurs when cos(θ) = -1. So we need

a2 - 3ab + 2b2 ≥ 0.

Factor:

a2 - 3ab + 2b2 = (a - b)(a - 2b).

Since we are already in the no-inner-loop case a > b, the factor a - b is positive. Therefore,

(a - b)(a - 2b) ≥ 0   if and only if   a - 2b ≥ 0.

Thus,

d2x/dy2 ≥ 0   if and only if   a/b ≥ 2.

Conclusion

When a/b ≥ 2, the left side of the limaçon does not bend inward. The curve may stretch, but it does not create an indentation.

Claim 1 also showed that r never becomes negative, so there is no inner loop. Together, these facts show that the limaçon is convex in the standard case a, b > 0.

Since rotations and reflections do not change convexity, the general condition is

|a/b| ≥ 2.

What you should notice: a controls the base circle-like size, while b controls how strongly the graph is pushed to one side. Convexity breaks when the pushing term becomes large enough to make the left side cave inward.
